# TaskList

## Basic Todolist App made in Golang
### To build it:
   
```
       git clone https://github.com/SyedAsadK/TaskList
    
       cd TaskList 
        
       go build ./cmd/todo
    
```
    

## To run the app ./todo in same directory

## Usage of TaskList:

  ```
  -add
    	add a new task
  -del int
    	deletes a task
  -done int
    	mark a task as completed
  -list
    	List all tasks

 ```
